# Text Rotate

Text Rotate ergänzt Google Docs™ um eine Funktion, die nativ fehlt: Text in Tabellen drehen oder vertikal anordnen.
Markieren Sie Text in einer Tabellenzelle, wählen Sie den gewünschten Winkel (z. B. 90°) oder den Vertikalmodus und prüfen Sie das Ergebnis in der Vorschau. Anschließend können Sie das Ergebnis direkt in Ihr Dokument einfügen.

Das Add-on übernimmt zentrale Formatierungen wie Schriftart, Schriftgröße, Textfarbe sowie Fett- und Kursivstil. Der gedrehte Text wird als Inline-Bild eingefügt. Mit „Bild → Text“ können Sie einen mit dem Add-on erzeugten Eintrag wieder in normalen, bearbeitbaren Text zurückwandeln.

So erstellen Sie klare Tabellenüberschriften und kompakte Layouts in Google Docs™ schneller und konsistent.
